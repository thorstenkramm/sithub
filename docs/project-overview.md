# Project Overview

## Project Name
SitHub

## Purpose
A desk booking web application for shared offices, enabling employees and admins to manage desks, rooms, and bookings efficiently.

## Executive Summary
SitHub provides real-time desk availability, notifications, and an admin dashboard. It supports Entra ID SSO and ships as a single binary with an embedded SQLite database and embedded frontend assets.

## Key Features (from README)
- Desk booking with flexible time ranges
- Interactive floor plan for availability
- Notifications for bookings and room activity
- Admin dashboard for managing areas/rooms/desks
- Entra ID SSO with group-based access
- Multi-language UI (EN/DE/ES/FR)

## Technology Stack Summary
- Go + Echo (backend)
- Vue 3 + Vuetify (frontend)
- REST API (JSON:API)
- SQLite3 (embedded)
- GitHub Actions (CI/CD, per README)
