# Release notes for version 0.0.3

## 🎉 New Features

- **Desktop weekly table view**: a new matrix view showing desks as rows and weekdays
  as columns, toggled via a switch on the item groups page; view preference is persisted
  per area
- **Inline booking from table cells**: click any free cell to open an anchored popover
  with self/colleague booking, inline note entry, and warning display — no navigation
  away from the matrix
- **Inline cancellation from table cells**: click your own booking (or any booking as
  admin) to cancel directly from the matrix with a lightweight confirmation popover
- **Note on booking create**: bookings can now carry a note from the moment of creation
  instead of requiring a separate edit step afterward
- **Floor plan booker names**: hover over occupied items on the floor plan to see the
  booker's full name in a tooltip; initials are shown when avatars are disabled

## 🔩 Improvements

- Date selector stays on the selected date after booking instead of resetting to today
- Reserved areas now show correct free/busy counts on floor plans (e.g. "3/4 free")
- Reserved items display visible cards with a lock badge instead of an opaque overlay;
  booker names and availability are now readable
- Week mode status chip shows real availability while a separate badge indicates
  reserved status, eliminating the duplicate "Reserved" label
- Avatar upload size limit raised from 512 KB to 4 MB to accommodate real Entra ID
  profile photos
- Sticky column and header in the table view for comfortable scrolling on wide matrices
- Room sections in the table view are collapsible with persisted state and
  occupied/total summary counts when collapsed

## 🐞 Bug Fixes

- Entra ID avatar sync failing with "not enough pixel data" on large response bodies
- Date picker jumping to today after every booking confirmation
- Double "Reserved" badge showing in week mode for reserved items
