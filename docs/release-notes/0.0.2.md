# Release notes for version 0.0.2

## 🎉 New Features

- **Interactive floor plans**: click on the floor plan to view availability, drill into
  subareas, and book directly from the map with free/busy color coding
- **Floor plan editor**: admin tool for positioning items on floor plans with drag, resize,
  and auto-save
- **Internationalization**: full i18n support with language selector and auto-detection;
  available in English, German, Spanish, French, and Ukrainian
- **Booking limits**: configurable maximum active bookings per user with clear error messages
- **User avatars**: profile photos synced from Entra ID; local users can upload their own
- **Reserved areas and items**: admins can mark areas and items as reserved, preventing
  regular users from booking them
- **Favorites**: mark frequently used item groups as favorites with free/busy indicators
  on the tiles
- **Equipment filter**: filter items by equipment on the item groups view with saved filter
  preferences
- **Week-mode booking**: book multiple days at once from a weekly view with per-day
  checkboxes and batch confirmation
- **Booking warnings**: configurable warnings on items with confirmation dialogs and
  "don't show again" option
- **Cancel from week view**: cancel individual day bookings directly from the week overview

## 🔩 Improvements

- Floor plan images fit the viewport on drill-down without scrolling
- Floor plan editor uses full page width with toolbar dropdowns instead of a sidebar
- Calendar widget starts on Monday
- Week selector shows the date range
- Custom icons configurable per area
- Profile and settings consolidated into a single menu
- Consistent snackbar confirmations across all actions
- Mobile layout improvements: text truncation, readable week tiles, floor plan touch support
- Memorized selected week and day across navigation
- Faster Entra ID login with async avatar sync and loading spinner

## 🐞 Bug Fixes

- Cancel booking dialog not closing after confirmation
- Heart icon visibility on booking tiles
- Floor plan full width on desktop
- SVG floor plans not displaying
- Connection lost error messaging
